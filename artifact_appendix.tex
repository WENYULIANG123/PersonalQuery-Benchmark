\section{Description}

PersonalQuery implements a grounded approach to generating personalized search queries from user reviews.

\textbf{Key Algorithms:}
\begin{itemize}
\item Dual Validation: Filters preferences using review evidence + metadata verification (71.5\% filtering rate)
\item Iterative Style Refinement: 16-dimensional linguistic feature matching
\item Spelling Difficulty Model: Cognitive difficulty scoring + user-specific error patterns
\end{itemize}
